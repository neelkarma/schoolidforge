_Note: This is only for reporting bugs. If you want to develop for School IDForge, see [DEVELOPERS.md](./DEVELOPERS.md)._

To report bugs, follow these steps:

1. **Before all, make sure that there isn't a duplicate of your issue already submitted in the [issues tab](../../issues/)!**
2. Make sure you're signed in to your GitHub account (if you don't have one, make one).
3. [Go to the issues tab of this repository.](../../issues/)
4. Click on the "New Issue" button.
5. Write a title that describes the bug in 1 sentence.
6. In the body of the issue, include:

- A detailed description of the bug
- Steps to reproduce the bug
- A screenshot if possible

Also remember, the more information I get, the faster I can release a bugfix!

7. To submit the bug report, click the "Submit New Issue" button.
8. I'll get to it when I have the time.
